Base URL: `https://api.orceum.com/v1`

All endpoints require `Authorization: Bearer YOUR_ORCEUM_TOKEN`.

---

## Register a Native App

<ParamField header="Authorization" type="string" required>
  Bearer token from your Orceum account
</ParamField>

```bash
POST /apps/
```

Creates a new native app and returns your `app_id` and `orceum_webhook_secret`.

**Request body:**

```json
{
  "name": "Todo Manager",
  "description": "Manage tasks with natural language",
  "app_type": "NATIVE",
  "auth_type": "API_KEY",
  "base_url": "https://todo.yourapp.com",
  "actions": [...],
  "config": { ... }
}
```

**Response `201`:**

```json
{
  "app_id": "app_a1b2c3d4",
  "name": "Todo Manager",
  "app_type": "NATIVE",
  "auth_type": "API_KEY",
  "status": "PENDING",
  "orceum_webhook_secret": "orc_sk_AbC123...",
  "created_at": "2024-01-15T12:00:00Z"
}
```

## Register an MCP App

```bash
POST /apps/mcp
```

Creates a new MCP app. Orceum connects to your server and runs `list_tools()` to auto-generate the manifest.

**Request body:**

```json
{
  "name": "GitHub Tools",
  "description": "Create issues and search repositories",
  "auth_type": "OAUTH",
  "mcp_server_url": "https://mcp.yourtool.com/mcp",
  "transport": "STREAMABLE_HTTP",
  "config": { ... }
}
```

**Response `201`:** Same structure as native app registration.

## List Your Apps

```bash
GET /apps/
```

Returns all apps owned by the authenticated developer.

**Response `200`:**

```json
{
  "apps": [
    {
      "app_id": "app_a1b2c3d4",
      "name": "Todo Manager",
      "status": "ACTIVE",
      "install_count": 142,
      "created_at": "2024-01-01T00:00:00Z"
    }
  ],
  "total": 1
}
```

## Get App Details

```bash
GET /apps/{app_id}
```

Returns full app details including config, actions, and status.

---

## Update an App

```bash
PATCH /apps/{app_id}
```

Update any app fields. All fields are optional — only provided fields are updated.

```json
{
  "description": "Updated description",
  "config": {
    "additional_usage_info": "New instructions for the AI.",
    "webhook_config": { "events_enabled": true }
  }
}
```

**Response `200`:** Updated app object.

---

## Delete an App

```bash
DELETE /apps/{app_id}
```

Permanently deletes the app and terminates all active installations. This cannot be undone.

<Warning>
Deleting an app immediately revokes all user installations without notification. Use the dashboard to archive instead if you may restore later.
</Warning>

**Response `204`: No content**

## Regenerate MCP Manifest

```bash
POST /apps/{app_id}/regenerate-manifest
```

Triggers Orceum to reconnect to your MCP server, run `list_tools()`, and update the manifest.

**Response `200`:**

```json
{
  "status": "ok",
  "tools_discovered": 8,
  "manifest_updated_at": "2024-01-15T12:00:00Z"
}
```

## Search Marketplace

```bash
GET /apps/marketplace/search
```

Search published apps.

| Query Param | Type | Description |
|-------------|------|-------------|
| `q` | string | Search query |
| `app_type` | string | `NATIVE` or `MCP` |
| `auth_type` | string | `NONE`, `API_KEY`, `OAUTH` |
| `limit` | integer | Results per page (max 100) |
| `offset` | integer | Pagination offset |

**Response `200`:**

```json
{
  "apps": [ ... ],
  "total": 42,
  "limit": 20,
  "offset": 0
}
```

## Get Marketplace Stats

```bash
GET /apps/marketplace/stats/{app_id}
```

Returns install counts and usage metrics for your app.

**Response `200`:**

```json
{
  "app_id": "app_a1b2c3d4",
  "active_installs": 1247,
  "total_installs": 1580,
  "uninstalls_last_30d": 34,
  "action_calls_last_30d": 18920
}
```
